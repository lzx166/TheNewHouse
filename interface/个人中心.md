##我发布的信息列表
## 请求
###	URL

+ 填写请求的URL

### POST /items
```json
{
    "pagesize": 10,
    "page": 1
}
```
### 必要项目

* `page`
* `pagesize`

### 默认项目
+ pagesize = `20`
* page = `1`

### 备注

* 默认列表按发布时间最新排序

## 响应

```json
{
    "ResultStatus": 200,
    "list": [
        {
            "itemid": "123",
            "title": "标题",
            "descripe": "描述",
            "price": "价格",
            "date": "日期",
            "area": "面积",
            "region": "区域",
            "houseid": "二手住宅，二手商铺对应的id",
            "layoutid": "户型的id",
            "img": "房屋照片"
        }
    ]
}
```

### `200` - 添加成功(目前：200)

### `400` - 请求参数错误
+ 其他错误由web端定义
